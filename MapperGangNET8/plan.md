#  график разработки проекта GangMapper по MVVM архитектуре

## Этап 1: Создание базовой структуры и UI-каркаса
**Цель:** Создать базовый каркас приложения с UI-элементами без функциональности

### Файлы для создания:
1. **Базовая структура проекта**
   - `App.xaml/App.xaml.cs` - Точка входа
   - `Infrastructure/DI/ContainerConfig.cs` - Базовая конфигурация DI
   - `Views/MainWindow.xaml` - Главное окно

2. **Базовые модели**
   - `Models/ActionEnum.cs` - Перечисление поддерживаемых действий
   - `Models/KeyBindingModel.cs` - Базовая модель для привязок клавиш

3. **Базовые ViewModels**
   - `ViewModels/ViewModelBase.cs` - Базовый класс для всех VM
   - `ViewModels/MainViewModel.cs` - Пустая VM для главного окна

4. **Базовая инфраструктура**
   - `Infrastructure/Commands/RelayCommand.cs` - Базовая реализация команд

### Тестирование:
- Запуск приложения и проверка отображения UI
- Проверка навигации между разделами (если есть)
